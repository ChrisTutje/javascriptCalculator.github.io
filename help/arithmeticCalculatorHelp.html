Calculator Help

Basic Operations

  • + - Addition
  • - - Subtraction
  • * - Multiplication
  • / - Division
  • = - Calculate result
  • A/C - Clear all
  • ⌫ - Delete last digit

Advanced Functions

  • ! - Factorial
  • √ - Square root
  • sin/cos/tan - Trigonometric functions
  • log/ln - Logarithm functions

Keyboard Shortcuts

  • Numbers 0-9 - Input digits
  • Enter - Calculate (=)
  • Escape - Clear (A/C)
  • Backspace - Delete (⌫)
